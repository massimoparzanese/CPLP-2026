## Consigna

¿Qué son los applets? ¿Qué son los servlets? (en JAVA)


## Respuesta


###### Applets

Los applets son pequeños programas escritos en Java que podían ejecutarse dentro de una página web, generalmente dentro de un navegador. Su objetivo era agregar funcionalidades interactivas a las páginas web, como animaciones, juegos simples, gráficos interactivos o simulaciones.

Los applets se descargaban automáticamente desde el servidor cuando el usuario abría la página web y se ejecutaban en el navegador mediante la Java Virtual Machine (JVM).

Sin embargo, con el paso del tiempo los applets dejaron de utilizarse debido a problemas de seguridad, rendimiento y compatibilidad con los navegadores modernos.


###### Servlets

Los servlets son componentes de Java que se ejecutan en el servidor y se utilizan para desarrollar aplicaciones web dinámicas. Funcionan dentro de un servidor web o de aplicaciones y se encargan de procesar las solicitudes de los clientes (por ejemplo, desde un navegador) y generar una respuesta, generalmente en formato HTML.

Un servlet recibe una petición HTTP, la procesa mediante código Java y devuelve una respuesta al cliente.

Los servlets forman parte de la tecnología utilizada para desarrollar aplicaciones web en Java y suelen trabajar junto con otras tecnologías como JSP (JavaServer Pages).
