1966 - 1970: APL, FORTRAN 66, BASIC, PL/I, SIMULA 67, ALGOL-W

Indique para cada uno de los períodos presentados cuales son las características nuevas que se

## Respuestas


#### Características por período

##### 1951–1955 — Lenguajes tipo Assembly

La característica principal fue la abstracción del código máquina: en lugar de escribir instrucciones en binario, los programadores podían usar mnemónicos (abreviaciones en inglés) que el ensamblador traducía a instrucciones de máquina. Assembly fue el primer lenguaje en usar instrucciones mnemónicas, lo que redujo el trabajo y facilitó la lectura, pavimentando el camino para los lenguajes de alto nivel.

##### 1956–1960 — FORTRAN, ALGOL 58, ALGOL 60, LISP
La gran novedad fue la aparición de los lenguajes de alto nivel con compilador. FORTRAN I introdujo nombres de hasta seis caracteres, bucles DO, I/O formateado, subprogramas definidos por el usuario y una instrucción de selección de tres vías. Por su parte, LISP fue el primer lenguaje funcional, orientado al procesamiento de listas y computación simbólica, con control mediante recursión y expresiones condicionales. ALGOL introdujo dos innovaciones clave: estructura de bloques anidados con scope léxico (variables privadas por bloque) y la notación BNF para describir formalmente la sintaxis de un lenguaje. También apareció la idea de portabilidad entre máquinas, algo inexistente hasta entonces.

##### 1961–1965 — COBOL, ALGOL 60, SNOBOL, JOVIAL
El foco pasó a la programación orientada a dominios específicos. COBOL originó conceptos como macros, estructuras de datos jerárquicas y nombres descriptivos; fue además el primer lenguaje cuyo uso fue mandado por el Departamento de Defensa de EE.UU. SNOBOL fue diseñado como lenguaje de manipulación de cadenas de texto, con potentes operadores para coincidencia de patrones (pattern matching). Se afianzó la idea de lenguajes especializados por dominio (científico, comercial, de texto).


#### 1966–1970 — APL, FORTRAN 66, BASIC, PL/I, SIMULA 67, ALGOL-W

Dos innovaciones fundamentales: tipado dinámico y la primera semilla de la programación orientada a objetos. APL se caracterizó por ser altamente expresivo con muchos operadores para escalares y arrays de varias dimensiones; tanto APL como SNOBOL se caracterizan por tipado dinámico y asignación dinámica de almacenamiento.El hito más importante fue SIMULA 67, diseñado para simulación de sistemas, que introdujo las clases como estructura que incluye datos locales y funcionalidad, siendo la base de la abstracción de datos, los objetos y la herencia. BASIC democratizó la programación al orientarse a usuarios no técnicos. PL/I fue el primer lenguaje en combinar programación concurrente y manejo de excepciones.


##### 1971–1975 — Pascal, C, Scheme, Prolog
El período de la programación estructurada y los sistemas operativos. Pascal enfatizó la programación estructurada rechazando el control de flujo arbitrario con GOTO; C fue un lenguaje de sistemas con tipos estáticos y facilidades para manipulación directa de memoria, usado como lenguaje de implementación de Unix; Prolog fue uno de los primeros lenguajes de programación lógica, donde los programas se expresan en relaciones en lugar de instrucciones; y Scheme introdujo el scope léxico al estilo ALGOL más la eliminación de llamadas recursivas en cola. 

##### 1976–1980 — Smalltalk, Ada, FORTRAN 77, ML
La consolidación de la programación orientada a objetos completa y el tipado estático funcional. Smalltalk fue un diseño completo de lenguaje orientado a objetos desde cero, donde todas las entidades del lenguaje son objetos, basado en el pasaje de mensajes y binding tardío.ML introdujo la inferencia de tipos Hindley-Milner, módulos y el estilo funcional con tipos estáticos, siendo ancestro de Haskell, F# y OCaml. Ada incorporó programación concurrente y tipado fuerte para software crítico.

##### 1981–1985 — Smalltalk 80, Turbo Pascal, PostScript
La novedad principal fue la masificación del desarrollo y la aparición de los entornos de desarrollo integrados (IDE). Smalltalk fue el primer IDE verdadero. Turbo Pascal de Borland revolucionó al ofrecer un compilador ultra-rápido y asequible para PCs. PostScript introdujo el concepto de lenguaje de descripción de páginas basado en gráficos vectoriales.

##### 1986–1990 — FORTRAN 90, C++, SML
La gran novedad fue la combinación de OOP con programación de sistemas. C++ evolucionó desde C y SIMULA 67, añadiendo facilidades para programación orientada a objetos y manejo de excepciones, soportando tanto programación procedural como OOP. FORTRAN 90 añadió soporte para recursión, punteros y tipos de datos definidos por el programador. SML estandarizó el paradigma funcional con tipos estáticos.

##### 1991–1995 — TCL, Perl, HTML
El período del scripting y la Web. La rápida expansión de Internet fue el siguiente gran evento histórico en los lenguajes de programación, abriendo una plataforma radicalmente nueva. Perl popularizó el procesamiento de texto y scripting de sistemas. Tim Berners-Lee desarrolló HTML en 1990 Computer Hope, que se masificó en estos años como el primer lenguaje de marcado para la web, no de programación tradicional sino de descripción de documentos hipertextuales.

##### 1996–2000 — Java, JavaScript, XML
La consagración de la programación orientada a la web y la portabilidad total. Java combinó ideas de C++ y Smalltalk con gestión automática de memoria, originalmente pensado para la web pero que dominó el software empresarial. JavaScript adoptó semántica similar a Scheme bajo sintaxis tipo Java, convirtiéndose en el lenguaje del frontend web. XML introdujo el concepto de metalenguaje de marcado extensible, para intercambio estructurado de datos entre sistemas.
