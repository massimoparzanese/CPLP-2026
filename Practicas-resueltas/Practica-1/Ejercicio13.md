## Consigna

Lenguaje - JavaScript

¿A qué tipo de paradigma corresponde este lenguajes? ¿A qué tipo de Lenguaje
pertenece?

## Respuesta


El lenguaje JavaScript es un lenguaje multiparadigma, ya que soporta diferentes estilos de programación.

##### Paradigmas

- Imperativo: permite definir secuencias de instrucciones que se ejecutan paso a paso.
- Orientado a objetos: utiliza objetos y prototipos para modelar datos y comportamiento.
- Funcional: permite el uso de funciones como valores, funciones de orden superior y callbacks.

#### Tipo de lenguaje

JavaScript es un:

- Lenguaje interpretado (aunque hoy en día también usa compilación JIT).
- Lenguaje de alto nivel.
- Lenguaje principalmente orientado al desarrollo web, tanto en el cliente (navegador) como en el servidor (por ejemplo con Node.js).
