### C

**Origen y contexto**. C es un lenguaje de propósito general creado en los años 70 por Dennis Ritchie. Fue desarrollado originalmente en Bell Labs entre 1972 y 1973 para construir utilidades que corrieran sobre Unix, y luego se usó para reimplementar el kernel del sistema operativo Unix. 
**Antecedentes**. Ken Thompson creó el lenguaje B en 1969-70, derivado directamente del BCPL de Martin Richards. Dennis Ritchie transformó B en C entre 1971 y 1973, manteniendo la mayor parte de la sintaxis de B pero agregando tipos de datos y muchos otros cambios, y escribiendo el primer compilador. 
**Consolidación**. En 1978, Brian Kernighan y Dennis Ritchie publicaron la primera edición de The C Programming Language, conocido como K&R, que sirvió durante muchos años como especificación informal del lenguaje. Luego, a partir de 1983, el comité ANSI X3J11 se encargó de estandarizar el lenguaje.
**Impacto**. C se convirtió en la base de la portabilidad de Unix: era más fácil mover el sistema operativo a nuevas máquinas porque no era necesario traducir todo manualmente al lenguaje ensamblador de cada plataforma. Durante los años 80 fue ganando popularidad gradualmente hasta convertirse en uno de los lenguajes más utilizados del mundo, con compiladores disponibles para prácticamente todas las arquitecturas modernas.

### Java

**Origen.** Java fue creado en Sun Microsystems, donde James Gosling lideró un equipo de investigadores con el objetivo de desarrollar un nuevo lenguaje que permitiera a los dispositivos electrónicos de consumo comunicarse entre sí. El trabajo comenzó en 1991, y antes de mucho tiempo el foco del equipo se desplazó hacia la World Wide Web. 
**El nombre.** El proyecto usó primero el nombre "Green" (internamente "Greentalk"), luego fue renombrado "Oak" inspirado en un roble que estaba afuera de la oficina de Gosling. Cuando Sun intentó registrar la marca "Oak", descubrieron que ya estaba reclamada por Oak Technologies, por lo que fue necesario un nuevo nombre.Gosling eligió finalmente el nombre "Java" —basado en un grano de café expreso— en 1995. 
**Lanzamiento y filosofía**. Java fue lanzado en mayo de 1995 como componente central de la plataforma Java de Sun. Prometía funcionalidad Write Once, Run Anywhere (WORA), ofreciendo entornos de ejecución gratuitos en las plataformas más populares. Los principales navegadores web incorporaron rápidamente la capacidad de ejecutar applets Java dentro de páginas web, y Java se volvió muy popular. 
**La innovación clave**. La diferencia entre Java y otros lenguajes fue revolucionaria: el compilador de Java transforma el código en bytecode, que luego es interpretado por la Máquina Virtual Java (JVM), actuando como un ordenador virtual que traduce el bytecode para cada máquina anfitriona. 
**Evolución.** En 1997 llegó JDK 1.1 con JDBC y RMI; en 1998 se introdujeron Java SE, Java EE y Java ME para distintas plataformas; en 2006 Sun liberó la JVM como software open source bajo la licencia GPL. Oracle, que compró Sun en 2010, es hoy el responsable de la implementación oficial de Java.
