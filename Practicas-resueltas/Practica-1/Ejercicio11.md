## Consigna

JAVA - RUBY - PHP
¿Qué tipo de programas se pueden escribir con cada uno de estos lenguajes? ¿A qué paradigma responde cada uno? ¿Qué características determinan la pertenencia a cada paradigma?

## Respuesta

### Python

El lenguaje Python permite desarrollar distintos tipos de programas, como:

- aplicaciones web

- scripts de automatización

- programas de análisis de datos

- inteligencia artificial y aprendizaje automático

- aplicaciones de escritorio

#### Paradigma

Python es un lenguaje multiparadigma, principalmente:

- orientado a objetos

- imperativo

- funcional

#### Características que determinan el paradigma

- Orientado a objetos: permite definir clases, objetos, herencia y encapsulamiento.

- Imperativo: el programa se ejecuta mediante una secuencia de instrucciones.

- Funcional: permite usar funciones como valores, funciones anónimas (lambda) y operaciones sobre colecciones.


### Ruby

El lenguaje Ruby se utiliza principalmente para:

- desarrollo de aplicaciones web

- scripting

- automatización de tareas

- desarrollo de prototipos de software

#### Paradigma

Ruby es un lenguaje orientado a objetos, aunque también admite un estilo imperativo y funcional.

#### Características que determinan el paradigma

- Todo en Ruby es un objeto.

- Permite definir clases y métodos.

- Soporta herencia y encapsulamiento.

- Permite trabajar con bloques y funciones de orden superior.

### PHP

El lenguaje PHP se utiliza principalmente para:

- desarrollo de aplicaciones web del lado del servidor

- generación dinámica de páginas web

- desarrollo de APIs web

#### Paradigma

PHP es un lenguaje multiparadigma, principalmente:

- imperativo

- orientado a objetos

- procedural

#### Características que determinan el paradigma

- Permite definir funciones y ejecutar instrucciones de manera secuencial (imperativo/procedural).

- Permite crear clases, objetos, herencia e interfaces (orientado a objetos).
