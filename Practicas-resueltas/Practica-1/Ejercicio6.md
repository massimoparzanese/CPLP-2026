## Consigna

Diga para qué fue, básicamente, creado Java.¿Qué cambios le introdujo a la Web?
¿Java es un lenguaje dependiente de la plataforma en dónde se ejecuta? ¿Por qué?

## Respuesta

### ¿Para qué fue creado Java?

Java fue creado originalmente por James Gosling en la empresa Sun Microsystems a comienzos de la década de 1990.
El objetivo inicial era desarrollar un lenguaje que permitiera crear software para dispositivos electrónicos y sistemas embebidos, como televisores interactivos y electrodomésticos inteligentes.

Posteriormente, Java comenzó a utilizarse ampliamente para el desarrollo de aplicaciones en Internet debido a su capacidad de ejecutarse en diferentes plataformas sin necesidad de modificar el código.

---

### ¿Qué cambios introdujo Java en la Web?

Java introdujo la posibilidad de crear aplicaciones interactivas dentro de las páginas web mediante los llamados applets. Estos pequeños programas podían ejecutarse dentro del navegador del usuario y permitían agregar funcionalidades dinámicas a los sitios web, como animaciones, gráficos interactivos o pequeños juegos.

Antes de Java, la mayoría de las páginas web eran principalmente estáticas. Con la aparición de Java, la Web comenzó a incorporar aplicaciones más complejas e interactivas.

---

### ¿Java es un lenguaje dependiente de la plataforma donde se ejecuta? ¿Por qué?

Java es considerado independiente de la plataforma. Esto se debe a que el código fuente del programa no se compila directamente a código máquina específico del sistema operativo, sino a un formato intermedio llamado bytecode.

Este bytecode se ejecuta sobre la Java Virtual Machine (JVM), que es un entorno de ejecución disponible para distintos sistemas operativos. De esta manera, el mismo programa puede ejecutarse en diferentes plataformas (Windows, Linux, macOS, etc.) sin necesidad de ser recompilado.

Esta característica se resume en el principio de Java:
“Write once, run anywhere” (escribir una vez, ejecutar en cualquier lugar).
